Automatch

Auto Matching is the process of matching PACS table record with RIS table record using different criteria. Both PACS and RIS hold patient information. As RIS is the Source of Truth, in other words Cleansing data, program needs mapping between PACS records so that while migration, mapped RIS data is used for cleansing.
